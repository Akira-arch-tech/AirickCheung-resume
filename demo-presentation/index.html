作品演示

每个演示对应仓库内 demo-presentation/<demo 名称>/ 目录，由 GitHub Actions 构建后发布到本站路径下。

  • wechat-ui-demo — 微信风格 UI（React + Vite，纯前端演示）
  • weichat-clone — 微信克隆项目演示（登录 / 会话 / 通讯录 / 朋友圈）
  • fundamental-custom-design-demo — 定制品设计网站 Demo（选品 + 定制 + 报价 + 下单）
  • FUN.DA.MENTAL-agent-mvp — 明星周边定制智能体 MVP（需求 -> 设计 -> 工单 -> 履约 -> 客服，含 requestId 链路追踪）

本地开发：进入对应子目录执行 npm install 与 npm run dev。
